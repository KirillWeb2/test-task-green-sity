# Панель мониторинга ёкаев

Система мониторинга духовных аномалий в режиме реального времени для Токио.

# Запуск

1. Клонируете репо
2. docker-compose build --no-cache
3. Перейдите на <a href="http://localhost:3000">http://localhost:3000</a> и радуйтесь результату :)

PS: Если возникнет ошибка `Cannot find module '/app/node_modules/next/dist/bin/next'`, удалите node_modules и запустите команду из пукнта 2
